# 변경 이력 (Changelog)

형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.1.0/)를 따르며, 버전은 [유의적 버전](https://semver.org/lang/ko/)을 따릅니다.

## [1.4.2] - 2026-07-19

### 변경됨
- **주간 히트맵의 정규 근무시간 점선이 분 단위까지 반영**된다(예: 8:30 시작이면 8시 칸 중간부터). 자정을 넘는 근무시간대(예: 22:00–06:00)도 두 구간으로 표시된다.

### 수정됨
- 휴식 시간 집계 정확도 개선: 점심시간 중 일시정지를 쓰면 그 시간이 이중으로 빠지던 문제와, 자정을 넘는 점심시간 설정 시 점심이 휴식에 잘못 합산되던 문제를 수정.
- 저녁에 앱을 켠 뒤 야간에 일을 시작하면 첫 "야간 근무 감지" 알림이 뜨지 않던 문제를 수정. (자정을 넘겨 계속 일할 때 감지 알림이 중복되지 않는 동작은 유지)
- 업데이트 확인 결과 창이 다른 창 뒤에 숨을 수 있던 문제를 수정(표시를 UI 스레드로 이동).
- 실근무·연속·휴식이 모두 10시간대인 날 트레이 툴팁이 글자 수 제한(63자)에 잘려 휴식 표시가 사라질 수 있던 문제를 수정(툴팁만 "7시간30분" 축약 표기).
- 데이터 폴더·설정 파일·다운로드 페이지 열기에 실패하면 아무 반응이 없던 것을 안내 풍선으로 알리도록 개선.

## [1.4.1] - 2026-07-19

### 변경됨
- **정시 퇴근 알림에 오늘 실근무 요약을 표시**한다("오늘 실근무 N시간 M분 — 수고하셨습니다!"). 또한 자리를 비운 사이에는 울리지 않고, 퇴근 시간 이후 **처음 활동할 때** 울리도록 바로잡았다(못 볼 알림을 소모하지 않음).

### 수정됨
- 데이터 파일(daily.csv 등)을 Excel 등 다른 프로그램이 잠근 상태로 열어 두면 저장 주기(60초)마다 오류 창이 뜰 수 있던 문제를 수정. 잠긴 동안 저장은 건너뛰고 다음 주기에 다시 시도하며, 시작 시 파일을 읽지 못해도 앱이 정상 실행된다.
- 트레이 메뉴 "설정 다시 읽기"가 설정 창 저장과 달리 알림 상태를 재평가하지 않던 문제를 수정. 이제 config.ini 를 직접 고쳐 다시 읽어도 이미 지난 조건의 알림이 뒤늦게 뜨지 않는다.

## [1.4.0] - 2026-06-29

### 추가됨
- **수면시간**: 설정한 시간대(기본 **00:00~07:00**)에는 모든 알림을 끈다(무음). 설정 창 "알림"에서 켜고 끄거나 시작·종료 시간을 바꿀 수 있으며 자정 넘김을 지원한다. 무음 중에도 내부적으로 알림 시점은 갱신되어, 깨어난 뒤 밀린 알림이 한꺼번에 뜨지 않는다.

### 변경됨
- **야간·초과근무·휴식 권유 알림을 1시간마다 반복**하도록 변경. (정시 퇴근·점심은 종전대로 하루 1회) 야간은 야간 실근무, 초과는 실근무, 휴식 권유는 연속근무 기준 1시간마다 다시 알린다.

### 수정됨
- 퇴근 후·야간에 PC를 켜면(앱 재시작) 지난 조건(정시 퇴근·야간·초과·점심)의 알림이 한꺼번에 뜨던 문제를 수정. 이제 알림은 앱이 켜져 있는 동안 조건이 새로 충족될 때만 뜬다.
- 자정을 넘겨 연속 근무할 때 휴식 권유 알림이 한동안 뜨지 않던 문제를 수정.

## [1.3.1] - 2026-06-29

### 변경됨
- **위장 모드(테스트 기능)를 끌 때도 확인 팝업을 표시**한다. 이제 트레이 메뉴에서 위장 모드를 켜거나 끌 때마다 항상 확인을 거친다. (켤 때의 책임 고지 동의 창은 그대로 유지)

## [1.3.0] - 2026-06-29

### 추가됨
- **업데이트 확인 (수동)**: 트레이 메뉴 "업데이트 확인" 으로 GitHub의 최신 릴리즈 버전을 조회해 현재 버전과 비교하고, 새 버전이 있으면 다운로드 페이지로 안내한다. 버튼을 누를 때만 동작하며(자동 확인 없음), 최신 버전 번호만 받아 올 뿐 사용자 데이터는 전송하지 않는다. 네트워크는 백그라운드로 처리되어 화면이 멈추지 않고, 인터넷이 없으면 안내 메시지만 표시하고 평소처럼 계속 동작한다.

## [1.2.0] - 2026-06-29

### 변경됨
- **휴식 표기를 '횟수'에서 '시간'으로 변경**: 오늘 현황·트레이 툴팁·주간 리포트·일일 마감 로그에서 휴식을 분 단위 누적 시간으로 표시한다. 근무 중 모든 자리비움(점심·일시정지 제외)을 합산하며, 근무 전후·야간의 유휴는 자동 제외된다. (데이터 파일에 휴식 산정용 열이 추가되었고, 이전 버전 파일과 자동 호환된다.)
- **위장 모드(테스트 기능) 켜기/끄기 풍선 알림 제거**: 상태는 트레이 툴팁의 `[위장]` 표시로만 안내한다. 위장 모드 동작과 켤 때의 책임 고지 동의 창은 그대로 유지된다.

### 수정됨
- 트레이 아이콘 생성 시 발생하던 아이콘 핸들 누수를 수정했다.

## [1.1.0] - 2026-06-27

### 추가됨
- **위장 모드 (테스트 기능)**: 활동 추정 방식의 한계를 점검하기 위한 실험 기능. 합성 마우스 입력으로 유휴 시간을 0으로 유지하며, 사람과 유사하게 곡선·가속/감속 궤적과 불규칙한 간격(짧은 간격 다수 + 가끔 긴 멈춤)으로 움직인다. 기본 비활성이며, 켤 때 책임 고지 및 테스트 기능 안내에 동의해야 작동하고 일시정지와 연동된다. 켜는 동안에는 이 앱의 측정값도 함께 영향을 받는다(의도된 동작). 본인 소유 PC에서의 사용을 전제로 한다.

## [1.0.1] - 2026-06-27

### 추가됨
- **Windows 시작 시 자동 실행** 옵션을 설정 창("기타")에 추가 (트레이 메뉴 토글과 동일 동작)

## [1.0.0] - 2026-06-27

첫 정식 릴리즈.

### 추가됨
- **워라밸 측정**: 실근무 / 시간외(정규시간 밖) / 야간 / 초과근무 자동 집계
- **점심시간 제외**: 설정한 점심 시간대는 실근무에서 제외(휴식으로 간주)
- **근무 요일**(기본 월~금): 비근무일에는 출·퇴근/점심 등 근무 직접 알림 미발생
- **연속근무 / 휴식** 추적 및 휴식 권유
- **주간 히트맵**: 요일 × 시각(0~23시)별 실근무량 시각화, 주 단위 이동
- **일·주 리포트**: 일일 마감 요약, 주간 리포트 자동·수동 생성, 워라밸 점수
- **알림 5종**(정시 퇴근·야간·휴식·초과·점심)과 **전체 알림 마스터 토글**
- **첫 실행 흐름**: 면책 조항 동의 → 근무시간 설정
- **설정 창**: 근무시간·점심·근무 요일·야간·측정·알림을 한 곳에서 편집(트레이 메뉴 공용)
- **트레이 메뉴**: 오늘 현황 · 주간 리포트 · 주간 히트맵 · 일시정지 · 시작 시 자동 실행 · 설정… · 데이터 폴더 열기 · 도움말 · 종료
- **로컬 저장**: 모든 데이터는 `%APPDATA%\IdleTimer\` 에만 저장(CSV/로그), 외부 전송 없음
- **저자원 단일 실행파일**: 별도 설치/런타임 불필요
